# 다꿍 다이어리 ❤️

## 1. 서비스 소개
- 손그림과 아기자기한 스티커들로 빼곡했던 아날로그 다이어리 감성은 다꿍-다이어리앱입니다.
- 여행다꿍, 맛집다꿍 등 원하는 주제로 원하는 내용을 사진, 손그림으로 꾸며낼 수 있습니다.

## 2. 차별점 
- 기존 다이어리는 너무 복잡하거나 단순해서 심심합니다.
- 심플하면서 효용성있는 디자인입니다.
